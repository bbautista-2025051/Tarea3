Actúa como un Arquitecto de Software Senior especializado en TypeScript, Node.js, Express, PostgreSQL y Angular (últimas versiones estables).

# Objetivo

Crear una aplicación para administrar clientes.

La aplicación únicamente debe permitir:

- Agregar clientes.
- Listar clientes.

No debe existir la funcionalidad de editar ni eliminar.

Todo el proyecto debe quedar listo para ejecutar inmediatamente después de instalar las dependencias.
